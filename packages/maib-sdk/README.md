# maib-sdk

> **Deprecated** — this package has been replaced by [`@maib/merchants`](https://github.com/zerodaysoft/maib-sdk).

## Migrate

```bash
npm uninstall maib-sdk && npm install @maib/merchants
```

Update your imports:

```typescript
import { ... } from "@maib/merchants";
```

Individual packages are also available: `@maib/checkout`, `@maib/ecommerce`, `@maib/rtp`, `@maib/mia`.
